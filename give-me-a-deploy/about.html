Before being able to spin up VMs from a GUI web application, Give-Me-A-Deploy (then called Give-Me-A-Vagrant) output deployable source code which could be downloaded and run with a few commands for a fully configured local VM. The process however was still slow and manual, and as they required manual termination, it did not solve the core problem of idle VMs.

Give-Me-A-Deploy is a GUI configurator which allows you to launch a customised VM in an environment of your choice. The VMs are set to be short-lived. They automatically expire after 48 hours (unless specified), cutting down on resource wastage due to idle VMs and making the deployment process as quick and hassle-free as possible.


Give-Me-A-Vagrant is brought to you by the Orion Continuous Delivery team.
