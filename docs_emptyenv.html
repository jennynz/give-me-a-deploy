This option on Give-Me-A-Vagrant simply outputs a Vagrantfile with a few parameters pre-configured for your convenience. It is very simple to boot, and you may add any provisioning you wish.

Booting up your VM
$ cd to/your/Vagrantfile/directory
$ vagrant up

Booting up your Vagrant VM simply requires you to move to the directory with your Vagrantfile in a command terminal


Installation System Requirements

In order for the scripts to run successfully, please install the following on your local machine:

  • VirtualBox
  • Vagrant
  • Git version control for cloning Stash repositories.

FAQ#1

The entire booting and provisioning process may take around 30 minutes depending on the speed of your host machine and internet connection. This is mostly due to the creation of the solution.zip, which takes about 20 minutes.
If you're impatient, check out the Vagrantfile and up the allocated memory.
