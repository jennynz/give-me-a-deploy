Documentation, FAQs & Help

Please select the type of virtual environment you are using:

  • Provisioned VM: on your own local machine.
  • DevStack: a local OpenStack server at Orion. IN DEV!
  • HP Cloud: a public cloud service used by Orion. IN DEV!
  • Empty VM: gives you just an empty machine with no provisioning.

This documentation can also be found on the Woki.
For the download functions on Give-Me-A-Deploy to work, we recommend Chrome or Firefox v.20+ that is HTML5 compatible.
