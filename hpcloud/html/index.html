Your provisioned VM is ready

Save these scripts to the same directory, $ chmod +x run.sh, then $ run.sh.

Download all three scripts in a zip

run.sh  is an executable bash shell script which kicks off the entire automated process of runing and provisioning your customised virtual machine. It clones Stash repositories, spins up a Vagrant VM and installs the OH products.

Vagrantfile  is the configuration file which allows run.sh to spin up the VM by simply calling vagrant up. It also calls install.sh as its provisioning shell script, to preload the newly created VM with the required dependencies and applications.

install.sh  is the shell provisioning script called by the Vagrantfile. It downloads and creates all the necessary files and directories for the installation, then automatically installs and starts the OH applications.
Once Platform and Portal have been installed and started, you can access them here:
http://localhost:19080/conductor
http://localhost:19080/concerto
