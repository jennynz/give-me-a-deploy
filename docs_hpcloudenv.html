A public cloud server such as HP Cloud is the most reliable solution to hosting VMs.
Eventually, this web application may be able to deploy the VM on HP Cloud straight from this GUI interface itself, instead of relying on an executable output source code. The VMs could also be timed to be short-lived, to reduce the number of idle VMs which are paid for by the number of hours per instance.
To see an example of a VM on HP Cloud, see Vagrant on HP Cloud.

Virtual Environment* "Provisioned Vagrant" checked. Provisioned Vagrant
VM Name* This hostname should contain only letters, numbers, hyphens or full stop. It can not start with a hyphen or full stop. portal-vm
Host Port* This only requires configuring if you have more than one VM running, where you would have to specify different host ports for each VM to avoid port collisions when setting up port forwarding. The guest port is set to a default of "19080" (this may be changed if desired in the output Vagrantfile). The host port is the port on your local host machine you would like forwarded to your guest machine. 19080
Memory* Customise the allocation of base memory to your VM. 768 MB
Products IN DEV! Which Orion Health application you would like pre-installed on your Vagrant machine. The code that you write in here is directly written out to a solutionVersion.yaml file which is used to create a solution.zip for the OHP installation, so it is important that the formatting is exactly as seen in the example shown e.g. indentation is two spaces, each application on a new line with the correct characterisation for version number. Portal 8.4.0.beta with Foundation 7.2.0.beta
Puppet Scripts IN DEV! Point us at your Puppet scripts in Subversion, and we'll provision the VM with your own configuration of Portal. http://subversion/src/Orchestral/Framework/PlatformBuild/trunk

FAQ#1

The entire booting and provisioning process may take around 30 minutes depending on the speed of your host machine and internet connection. This is mostly due to the creation of the solution.zip, which takes about 20 minutes.
If you're impatient, check out the Vagrantfile and up the allocated memory.
