Give-Me-A-Vagrant is a GUI configurator for the Vagrant automation tool. It provisions a VM with an operating system of your choice, and is able to also preload it with Orion Health products using Puppet.

Next: Give-Me-A-Deploy

Even better than deployable source code would be pre-configured VMs which can be spun up in a local (OpenStack) or cloud server (HP Cloud) and destroyed from a GUI interface. These short-lived VMs would eliminate the problem of resource wastage due to idling, and provide an efficient solution to the demand for fast and flexible development environments.


Give-Me-A-Vagrant is brought to you by the Orion Continuous Delivery team.
